# Git Basics to Advanced 🚀

## 📌 Overview
This repository demonstrates fundamental to advanced Git operations including local and remote repository management, branching, merging, and version control workflows.

---

## 🎯 Objectives
- Understand Version Control System (VCS)
- Perform Git operations (init, add, commit, push, pull)
- Work with branches and merging
- Manage local and remote repositories

---

## 🧠 Key Concepts
- Distributed Version Control System (DVCS)
- Working Directory, Staging Area, Repository
- Branching and Merging
- Remote Repository (GitHub)

---

## ⚙️ Tools Used
- Git
- GitHub
