# 🧪 Basic Git Demo

---

## 📌 Overview

This demo showcases a **simple Git workflow** from initializing a repository to pushing code to GitHub.

👉 Perfect for beginners to understand the basic Git flow.

---

## 🎯 Objective

Learn how to:
- Initialize a Git repository  
- Add and commit changes  
- Connect to a remote repository  
- Push code to GitHub  

---

## 🚀 Step-by-Step Demo

---

### 1️⃣ Create Project Folder

```bash
mkdir basic-demo
cd basic-demo
2️⃣ Initialize Git
git init
3️⃣ Create a File
echo "# Basic Demo" > README.md
4️⃣ Check Status
git status
5️⃣ Add Files
git add .
6️⃣ Commit Changes
git commit -m "Initial commit"
7️⃣ Add Remote Repository
git remote add origin <repository-url>
8️⃣ Push to GitHub
git push -u origin main
🔄 Workflow Summary
git init → git add → git commit → git push
🧠 Key Learnings
Git tracks file changes
Commits save project history
Remote repositories enable sharing
Push uploads your code
⚠️ Common Mistakes
Forgetting git add
Not setting remote
Wrong branch name (main vs master)
🎯 Summary

This demo helps you understand the basic Git lifecycle and prepares you for more advanced workflows.
